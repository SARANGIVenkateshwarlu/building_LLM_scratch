```

### 4.3. `vocab.bpe` → BPE Rules

```

## 5. Weight Mapping: TensorFlow → PyTorch

**Challenge**: OpenAI saved **TensorFlow format**, we built **PyTorch model**.

**Solution**: `load_gpt2_params_from_tf_checkpoint()` → **params dict** with **5 keys**:

```
params = {
  "wte": ,     # Token embeddings
  "wpe": ,     # Positional embeddings  
  "blocks": [12 blocks × {attn, ffn, ln, ...}],  # All Transformer blocks
  "ln_f.g": ,         # Final LayerNorm scale
  "ln_f.b":           # Final LayerNorm bias
}
```


### 5.1. Architecture Mapping

```
Our GPT architecture:
TokenEmbeds ──+── PosEmbeds ── Dropout ── Transformer ── LN ── LMHead[^1]
              │
OpenAI names: wte           wpe                    blocks[*]    ln_f.g/b

Transformer block:
  LN1 ── MultiHeadAttn(QKV) ── + ── LN2 ── FFN ── + ── OutputProj
  g/b        c_attn.w/b           g/b     fc.w/b    c_proj.w/b
```

### Step 7.2. Load Weights (Custom Function)

```python
def load_gpt2_weights(model, params):
    # 1. Token + Positional embeddings
    model.token_embedding.weight.data = torch.tensor(params['wte'])
    model.position_embedding.weight.data = torch.tensor(params['wpe'])
    
    # 2. Transformer blocks (12 layers)
    for i in range(model.config.n_layers):
        block = model.transformer.blocks[i]
        
        # Attention QKV (fused → split)
        fused_qkv = torch.tensor(params['blocks'][i]['attn']['c_attn.w'])
        block.attention.W_Q.data = fused_qkv[:768, :768]
        block.attention.W_KV.data = fused_qkv[768:, :]
        
        # FFN
        block.feed_forward.fc1.weight.data = torch.tensor(params['blocks'][i]['ffn']['fc.w'])
        block.feed_forward.fc2.weight.data = torch.tensor(params['blocks'][i]['ffn']['proj.w'])
        
        # LayerNorms
        block.ln1.gamma.data = torch.tensor(params['blocks'][i]['ln1.g'])
        block.ln1.beta.data  = torch.tensor(params['blocks'][i]['ln1.b'])
        block.ln2.gamma.data = torch.tensor(params['blocks'][i]['ln2.g']) 
        block.ln2.beta.data  = torch.tensor(params['blocks'][i]['ln2.b'])
    
    # 3. Final LayerNorm + LM Head
    model.ln_f.gamma.data = torch.tensor(params['ln_f.g'])
    model.ln_f.beta.data  = torch.tensor(params['ln_f.b'])
    model.lm_head.weight.data = torch.tensor(params['wte'])  # Weight tying!

load_gpt2_weights(model, params)
model.eval()
```
